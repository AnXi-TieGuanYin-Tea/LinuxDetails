# 1.mmap_write_performance  

## 问题  

对比mmap和write的性能，哪个性能更好？

## 结论  

利用mmap写文件的性能不如直接写文件。

## Question 

Compare the performance of mmap and write, which is better?

## Answer  

The performance of mmap write is worser than direct write.
